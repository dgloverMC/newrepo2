How to create a new customer

Customers have accounts with a profit center as a way to keep track of purchases for different jobs, consistent pricing, as well as expediting invoices.  The Customer function may be used to set up a new account, maintain an existing account (or copy an existing account for even faster set up) or delete an account.

Enter a Customer ID and Customer Name.  To create more consistent Customer IDs, please refer to the Recommended SPEAKS Guidelines.

F8100 Customer and Accounts File Screen Description

Each customer with open account privileges must be set up in CEDNet.  Although a variety of information is required for each customer, some of the information is optional and does not need to be entered in CEDNet.

Non Standard Tool Strip Functions

These functions are found at the top of the Account Maintenance screen and are only available in this screen.

Field Description

Sh F7 New Customer

Press Sh+F7 or click Sh F7 New Customer to create a new customer.  The message ** IN ADD MODE** appears next to the Customer Name field.

Sh F12 Delete Customer

Press Sh+F12 or click Sh+F12 Delete to delete the selected customer.
Header

Field

Description

Customer ID

The Customer ID is a profit center defined code.  It is used to link customer accounts together in order to facilitate adding customers to SPA Agreements as well as linking customer job accounts together.

Account #

This field is a five-digit customer number assigned by the Credit Office.

Customer Name

Enter a Customer Name if creating a new customer; otherwise this field is seeded by an existing customer name. Up to 35 characters can be entered.

Trade Partner ID

This is an EDI field specific to customers who transmit information via EDI.

Notes

Notes pertaining to this customer can be entered here if needed.  These notes are also displayed on sales registers during the entry process.

Enable PDF Invoices as a Single Attachment

When checked, all PDF invoices for each account under the selected customer will be combined into a single PDF attachment and the 'Number of Fax/Emails to Send' will be set to one per recipient-all invoices attached.

Enable 3rd-Party Invoicing

When checked, invoices for all accounts for the customer where Send By on the 2-Billing tab of the F8110 Account Maintenance screen is set to Email, Fax ,or US Mail, will be forwarded to the 3rd-Party billing service provider for distribution to the customer.

Note:  Use the Apply to all accounts button on the 4-Defaults and Flags>invoicing tab of the F8901 Home Profit Center screen to quickly mark this checkbox for all customers.

 

Accounts Tab

This tab shows the accounts associated with the specific Customer Name entered.

Field Description

Select

 

Status

Type

Displays the account type as either Shop or Job.

Acct #

Displays the account number as maintained in F8110 Account Maintenance.

Account Name

Displays the account name as maintained in F8110 Account Maintenance.

Credit Status

Displays the account's current credit status which is determined by your Profit Center's Credit Office.

Ship Address Line 1

Displays the default ship to address as maintained on the F8110 Account Maintenance screen Shipping tab.

Ship Address Line 2

Displays additional address information as needed.

Ship City

Displays ship to city.

Ship State

Displays ship to state.

Ship Zip

Displays zip code for the ship to city/state.

Enable 3rd Party Invoicing

Mark this check box to activate 3rd party invoice delivery service for this account.

Grid Specific Buttons

Field Description

Field

Description

New Account F7

Press F7 or click New Account F7 to open the F8110 Account Maintenance screen to create a new customer account.

Edit Account F8

Press F8 or click Edit Account F8 to open the F8110 Account Maintenance screen and edit account information.

Copy Account F9

Press F9 or click Copy Account F9 to copy the currently highlighted account.  The copy process takes information from an existing account number to a new account number.

Delete Account F12

Press F12 or click Delete Account F12 to delete the currently highlighted account.  If any open documents exist for this account, the account can not be deleted.

Note: If you delete an account, the account cannot be restored, but the account number can be re-used for a new account.

F8151 Customer Search Screen Description

The F8151 Customer Search screen opens automatically when you select Customer and Accounts File from the Maintenance>Customer menu.

Non Standard Tool Strip Functions

These functions are found at the top of the Account Maintenance screen and are only available in this screen.

Field Description

F7 New

Press F7 or click F7 New to open the F8100 Customer and Accounts File screen and create a new customer.  The message ** IN ADD MODE** appears next to the Customer Name field.

 

F8151 Customer Search

Field Description

Customer ID

Enter part or all of a Customer ID to help refine the search results.

Go To Account #

Select an account from the drop down to go directly to the F8110 Account Maintenance screen.

Name

Enter part or all of a customer name to help refine the search results.

Search Results

Customer ID/Name

Displays all customer IDs and their name that match the search criteria.  When the screen is first opened, all customer IDs and names will be displayed.

How to create a new customer

·         Select Maintenance>Customer>Customer and Accounts File to open the F8151 Customer Search screen.  

o        All Customer IDs will be automatically displayed in the Search Results grid.  To narrow the results, enter as much information known for a Customer ID or Account.

 

·         To create a new customer from this screen press F7 or click F7 New to open the F8100 Customer and Accounts File screen.

·         Enter the new Customer ID.

·         Enter the new Customer Name.

·         Enter any specific customer notes as needed.

·         F2 Save.

 

Note: You can also create a new customer while viewing an existing customer in the F8100 Customer and Account File screen by pressing Shift+F7 or click Sh F7 New Customer.
